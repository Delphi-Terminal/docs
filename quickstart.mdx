## 1. Try it in the playground

A test API key is already pre-filled in the docs. Head to the **API Reference** tab, pick any endpoint, and hit **Try it** — no signup or key setup required.

## 2. Make a request from your terminal

You can also use the test key directly with `curl`:

```bash
curl -X GET 'https://api.delphiterminal.co/api/v1/klsi/markets' \
  -H 'X-API-Key: YOUR_API_KEY'
```

Or search across all markets:

```bash
curl -X GET 'https://api.delphiterminal.co/api/v1/events/search?q=bitcoin' \
  -H 'X-API-Key: YOUR_API_KEY'
```
